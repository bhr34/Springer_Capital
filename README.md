
# Springer Capital Data Analysis

This repository contains the Python script, CSV datasets, data profiling reports, and data dictionary for Springer Capital analysis.

---

## Requirements

- Python 3.11 (or compatible version)  
- Libraries: pandas, scikit-learn, openpyxl  
- Docker (optional, if you want to run in a container)

---

## Files in this repository

- `springer-capital-son.py` → Python analysis script  
- `data_dictionary.xlsx` → Data dictionary for all CSV files  
- `profiling_reports/` → Folder containing profiling HTML reports  
- CSV files: `lead_log.csv`, `user_referrals.csv`, etc.

---

## Run Python script

1. Open terminal or command prompt.  
2. Navigate to the folder containing the script and CSV files.  
3. Run the script:

```bash
python springer-capital-son.py
